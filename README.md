### Download Repo

```cmd
git clone 
```







### Install Dependencies

```cmd
composer i 
```







### Create DB Name 

- `Create DB name with mmcovid-19`

  ```.env
  DB_CONNECTION=mysql
  DB_HOST=127.0.0.1
  DB_PORT=3306
  DB_DATABASE=mmcovid-19
  DB_USERNAME=root
  DB_PASSWORD=
  ```







### Migrate

```cmd
php artisan migrate
```

> Now you can see app in your browser (http://localhost/mmcovid-19/public/)
